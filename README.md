# to_do_list_project
A Python console-based app to manage daily tasks using lists.

### 🔧 Features:
- ➕ Add Task  
- 📋 View All Tasks  
- ❌ Delete Any Task  
- 🗑 Clear All Tasks  
- 🔚 Exit the app

### 💻 Technologies:
- Python (List, Functions, Input/Output, Loops)
